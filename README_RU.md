# HabaNada

*Гибридный блокировщик рекламы и ИИ-инструмент для переписывания кликбейта на базе локальной модели Gemini Nano в Chrome.*

## Обзор

HabaNada — это браузерное расширение нового поколения для Google Chrome, объединяющее высокопроизводительный блокировщик сетевых запросов и семантическую фильтрацию контента. В отличие от стандартных блокировщиков рекламы, которые полагаются исключительно на статические списки фильтрации, HabaNada использует встроенный локальный искусственный интеллект (Gemini Nano) для классификации нативной рекламы и динамического переписывания кликбейтных заголовков в нейтральный информативный текст. Все вычисления ИИ выполняются локально на вашем устройстве, гарантируя конфиденциальность данных и скорость работы.

---

## Основные возможности

- **Гибридная блокировка рекламы** — сочетание правил Declarative Net Request для блокировки известных рекламных сетей с динамическим скрытием пустых рекламных блоков (схлопыванием верстки) через инжектируемые CSS-стили.
- **Защита от систем обхода блокировок** — синхронный перехват и блокировка анти-адблок скриптов (таких как AdShield и OptiDigital) в основном мире (Main World) до отправки сетевого запроса браузером, а также автоматическое скрытие предупреждающих баннеров с разблокировкой скролла.
- **Блокировка уведомлений и раздражителей (Annoyances)** — перехват нативного API браузера для запроса push-уведомлений и скрытие навязчивых всплывающих окон подписки на рассылки с использованием алгоритмов поиска родительских контейнеров.
- **Локальный ИИ для борьбы с кликбейтом** — использование встроенного в Chrome API `window.ai.languageModel` для классификации и переписывания сенсационных кликбейтных заголовков.
- **Независимое управление функциями** — раздельные тумблеры управления блокировкой рекламы (для каждого сайта) и антикликбейтом (глобально), позволяющие настраивать их независимо.
- **Мультиязычность** — полностью локализованный интерфейс всплывающего окна (popup), автоматически определяющий язык интерфейса браузера и поддерживающий русский, английский, немецкий, испанский и французский языки.

## Быстрый старт

### Требования

Для работы HabaNada вам потребуется Google Chrome версии 128 или новее с активированным встроенным ИИ.

1. Перейдите по адресу `chrome://flags/#optimization-guide-on-device-model` и выберите **Enabled BypassPerfRequirement**.
2. Перейдите по адресу `chrome://flags/#prompt-api-for-sharing` и выберите **Enabled**.
3. Перезапустите Google Chrome.
4. Перейдите по адресу `chrome://components/` и найдите компонент **Optimization Guide On Device Model**. Нажмите кнопку **Проверить обновления** для скачивания модели ИИ.

### Установка

1. Клонируйте или скачайте этот репозиторий на локальный компьютер.
2. Откройте Google Chrome и перейдите на страницу управления расширениями `chrome://extensions/`.
3. Включите **Режим разработчика** с помощью переключателя в правом верхнем углу.
4. Нажмите кнопку **Загрузить распакованное расширение** в левом верхнем углу.
5. Выберите корневую папку репозитория, содержащую файл `manifest.json`.

## Разработка и архитектура

Расширение состоит из пяти основных файлов:

- `manifest.json` — конфигурационный файл, декларирующий разрешения, фоновый скрипт и параметры запуска контентных скриптов.
- `background.js` — фоновый сервис-воркер, управляющий белым списком доменов, динамическими правилами DNR и хранящий статистику.
- `content.js` — контентный скрипт, запускаемый на этапе `document_start`. Отвечает за блокировку AdShield-скриптов, инжекцию стилей, кэширование результатов классификации ИИ для избежания повторных запросов на SPA-страницах и инициализацию MutationObserver.
- `bridge.js` — скрипт, внедряемый в основной мир страницы (Main World) для прямого доступа к API `ai.languageModel`, выполняющий классификацию текстов.
- `rules.json` — статический набор правил declarativeNetRequest для блокировки сетевых запросов популярных рекламных провайдеров.
